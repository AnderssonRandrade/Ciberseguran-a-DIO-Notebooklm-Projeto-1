# Caderno Temático com NotebookLM – Fundamentos de Cibersegurança

## Objetivo do Projeto

Este projeto foi desenvolvido como desafio prático da DIO com o objetivo de utilizar Inteligência Artificial como ferramenta de aprendizagem ativa através do NotebookLM.

O tema escolhido foi **Fundamentos de Cibersegurança**, devido à relevância da área para o mercado de tecnologia e também por fazer parte dos meus objetivos profissionais.

Objetivos específicos:

* Compreender conceitos fundamentais de segurança da informação;
* Identificar ameaças digitais comuns;
* Entender métodos de proteção e defesa;
* Criar material reutilizável para revisões futuras;
* Desenvolver habilidades de engenharia de prompts utilizando IA.

## Engenharia de Prompts e Testes

### Prompt 1

"Explique os pilares da Segurança da Informação utilizando exemplos práticos."

Resultado:

A IA apresentou os três pilares principais:

* Confidencialidade
* Integridade
* Disponibilidade

Exemplo:

Confidencialidade → senha bancária protegida.

---

### Prompt 2

"Crie uma explicação simples sobre ataques de phishing para iniciantes."

Resultado:

A IA apresentou uma definição clara, exemplos de e-mails falsos e formas de prevenção.

---

### Prompt 3

"Compare malware, ransomware e spyware em formato de tabela."

Resultado:

A resposta permitiu visualizar diferenças rapidamente.

---

### Dificuldades encontradas ("Cicatrizes")

Problema:

As respostas iniciais estavam excessivamente técnicas.

Solução:

Foram adicionadas instruções como:

* "Explique como se estivesse ensinando um iniciante";
* "Utilize exemplos práticos";
* "Resuma em tópicos".

Resultado:

As respostas tornaram-se mais claras e objetivas.

# Miniguia de Estudos

## Resumo Estruturado

### O que é Cibersegurança?

Cibersegurança é o conjunto de práticas, ferramentas e processos utilizados para proteger sistemas, redes, dados e dispositivos contra ataques digitais.

### Principais ameaças:

* Phishing
* Malware
* Ransomware
* Spyware
* Engenharia Social
* Ataques DDoS

### Boas práticas:

* Utilizar autenticação de dois fatores;
* Manter sistemas atualizados;
* Utilizar senhas fortes;
* Evitar links suspeitos;
* Fazer backups periódicos.

---

## Glossário

**Phishing:** Técnica usada para enganar usuários e roubar informações.

**Malware:** Software malicioso criado para causar danos.

**Firewall:** Sistema que controla tráfego de rede.

**VPN:** Tecnologia que cria conexão segura entre dispositivos.

**Criptografia:** Método que transforma informações em códigos protegidos.

**Autenticação Multifator (MFA):** Camada extra de segurança além da senha.
